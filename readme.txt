Preloading Page

== Description ==

An Easy to use Pre Loading page which will help you in many ways like adding age confirmation, making youser confirm terms or need to show anything to users... It will also helps you increase your Email subscriptins.

Features include:

* Create Splash Page for your Website.
* You can add Age Verification.
* Support Full Screen and Normal Screen Video embedding.
* Opt In form for accepting terms to get in.
* Manage Blocks background and text colors.
* Totall Customisable as per your needs.

**Are you missing something or isn't it working as expected ? I am open to suggestions to improve the plugin !**

== Installation ==

Activate the plugin.

You can set everything right from the Admin panel of Plugin.
